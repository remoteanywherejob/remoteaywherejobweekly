## 소개

[Remote Anywhere Job](https://www.remoteanywherejob.com)은 전 세계에서 원격 소프트웨어 채용 정보를 수집하는 플랫폼입니다.

이 저장소는 매주 최신 원격 소프트웨어 채용 공고를 자동으로 집계하여 카테고리별로 구성하여 쉽게 탐색하고 검색할 수 있습니다.

## 채용 카테고리

- 프론트엔드 개발자
- 백엔드 개발자
- 풀스택 개발자
- 모바일 개발자
- DevOps/SRE
- 데이터 엔지니어/데이터 과학자
- ML/AI 엔지니어
- 제품 매니저
- 디자이너
- QA/테스트

## 사용 방법

1. **채용정보 탐색**: 특정 날짜 폴더로 이동하여 보고서 보기
2. **채용 찾기**: 채용 제목 클릭하여 원래 공고로 직접 이동
3. **카테고리별 필터링**: 빠른 탐색을 위해 채용이 카테고리별로 그룹화됨

## 업데이트 주기

- 매주 월요일 자동 업데이트
- 업데이트마다 새로운 보고서 파일 생성
