## 關於

[Remote Anywhere Job](https://www.remoteanywherejob.com) 是一個收集全球遠端軟體工作機會的平台。

本倉庫每週自動彙總最新的遠端軟體工作招聘資訊，按類別整理，方便瀏覽和搜尋。

## 工作類別

- 前端開發工程師
- 後端開發工程師
- 全端開發工程師
- 行動端開發工程師
- DevOps/SRE
- 資料工程師/資料科學家
- ML/AI 工程師
- 產品經理
- 設計師
- QA/測試

## 使用方法

1. **瀏覽工作**：進入對應日期的資料夾查看報告
2. **尋找工作**：點擊工作標題可直接跳轉到原始招聘資訊
3. **按類別篩選**：工作按類別分組，方便快速導航

## 更新頻率

- 每週一自動更新
- 每次更新生成新的報告檔案
