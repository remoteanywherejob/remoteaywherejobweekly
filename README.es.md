## Acerca de

[Remote Anywhere Job](https://www.remoteanywherejob.com) es una plataforma que recopila trabajos de software remotos de todo el mundo.

Este repositorio agrega automáticamente las últimas publicaciones de trabajos de software remotos cada semana, organizadas por categoría para facilitar la navegación y búsqueda.

## Categorías de Trabajos

- Desarrollador Frontend
- Desarrollador Backend
- Desarrollador Full Stack
- Desarrollador Mobile
- DevOps/SRE
- Ingeniero de Datos/Científico de Datos
- Ingeniero ML/AI
- Gerente de Producto
- Diseñador
- QA/Testing

## Cómo Usar

1. **Explorar Trabajos**: Navega a la carpeta de una fecha específica para ver el informe
2. **Encontrar Trabajos**: Haz clic en el título del trabajo para ir directamente a la publicación original
3. **Filtrar por Categoría**: Los trabajos están agrupados por categoría para una navegación rápida

## Frecuencia de Actualización

- Actualizado automáticamente cada lunes
- Se genera un nuevo archivo de informe con cada actualización
