## About

[Remote Anywhere Job](https://www.remoteanywherejob.com) is a platform that collects remote software jobs from around the world.

This repository automatically aggregates the latest remote software job postings every week, organized by category for easy browsing and searching.

## Job Categories

- Frontend Developer
- Backend Developer
- Full Stack Developer
- Mobile Developer
- DevOps/SRE
- Data Engineer/Scientist
- ML/AI Engineer
- Product Manager
- Designer
- QA/Testing

## How to Use

1. **Browse Jobs**: Navigate to the folder for a specific date to view the report
2. **Find Jobs**: Click on a job title to go directly to the original posting
3. **Filter by Category**: Jobs are grouped by category for quick navigation

## Update Frequency

- Automatically updated every Monday
- A new report file is generated with each update
