## について

[Remote Anywhere Job](https://www.remoteanywherejob.com) は、世界中からリモートソフトウェア求人を収集するプラットフォームです。

このリポジトリでは、毎週最新のリモートソフトウェア求人募集を自動的に集約し、カテゴリー別に整理して閲覧と検索を容易にしています。

## ジョブカテゴリー

- フロントエンド開発者
- バックエンド開発者
- フルスタック開発者
- モバイル開発者
- DevOps/SRE
- データエンジニア/データサイエンティスト
- ML/AIエンジニア
- プロダクトマネージャー
- デザイナー
- QA/テスト

## 使い方

1. **求人を閲覧**: 特定の日付のフォルダに移動してレポートを表示
2. **求人を探す**: 求人のタイトルをクリックして元の募集に直接アクセス
3. **カテゴリーでフィルター**: 求人はクイックナビゲーションのためカテゴリー別にグループ化

## 更新頻度

- 毎週月曜日自動的に更新
- 更新ごとに新しいレポートファイルを生成
