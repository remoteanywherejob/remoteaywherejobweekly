## Over

[Remote Anywhere Job](https://www.remoteanywherejob.com) is een platform dat remote software banen verzamelt van over de hele wereld.

Deze repository aggregeert automatisch de nieuwste remote software vacatures elke week, georganiseerd per categorie voor eenvoudig browsen en zoeken.

## Vacaturecategorieën

- Frontend Ontwikkelaar
- Backend Ontwikkelaar
- Full Stack Ontwikkelaar
- Mobile Ontwikkelaar
- DevOps/SRE
- Data Engineer/Data Scientist
- ML/AI Ingenieur
- Product Manager
- Designer
- QA/Testing

## Hoe Te Gebruiken

1. **Vacatures Bladeren**: Navigeer naar de map voor een specifieke datum om het rapport te bekijken
2. **Vacatures Vinden**: Klik op een vacaturetitel om direct naar de originele vacature te gaan
3. **Filter Op Categorie**: Vacatures zijn gegroepeerd per categorie voor snelle navigatie

## Update Frequentie

- Automatisch bijgewerkt elke maandag
- Een nieuw rapportbestand wordt gegenereerd bij elke update
