## 关于

[Remote Anywhere Job](https://www.remoteanywherejob.com) 是一个收集全球远程软件工作机会的平台。

本仓库每周自动汇总最新的远程软件工作招聘信息，按类别整理，方便浏览和搜索。

## 工作类别

- 前端开发工程师
- 后端开发工程师
- 全栈开发工程师
- 移动端开发工程师
- DevOps/SRE
- 数据工程师/数据科学家
- ML/AI 工程师
- 产品经理
- 设计师
- QA/测试

## 使用方法

1. **浏览工作**：进入对应日期的文件夹查看报告
2. **查找工作**：点击工作标题可直接跳转到原始招聘信息
3. **按类别筛选**：工作按类别分组，方便快速导航

## 更新频率

- 每周一自动更新
- 每次更新生成新的报告文件
