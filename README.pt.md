## Sobre

[Remote Anywhere Job](https://www.remoteanywherejob.com) é uma plataforma que reúne empregos de software remotos de todo o mundo.

Este repositório agrega automaticamente as últimas publicações de empregos de software remotos todas as semanas, organizadas por categoria para fácil navegação e pesquisa.

## Categorias de Empregos

- Desenvolvedor Frontend
- Desenvolvedor Backend
- Desenvolvedor Full Stack
- Desenvolvedor Mobile
- DevOps/SRE
- Engenheiro de Dados/Cientista de Dados
- Engenheiro ML/AI
- Product Manager
- Designer
- QA/Teste

## Como Usar

1. **Procurar Empregos**: Navegue até a pasta de uma data específica para ver o relatório
2. **Encontrar Empregos**: Clique no título do emprego para ir diretamente para a publicação original
3. **Filtrar por Categoria**: Os empregos são agrupados por categoria para navegação rápida

## Frequência de Atualização

- Atualizado automaticamente toda segunda-feira
- Um novo arquivo de relatório é gerado a cada atualização
