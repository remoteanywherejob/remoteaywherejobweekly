## Über

[Remote Anywhere Job](https://www.remoteanywherejob.com) ist eine Plattform, die Remote-Softwarejobs aus aller Welt sammelt.

Dieses Repository aggregiert automatisch die neuesten Remote-Softwarejobangebote jede Woche, kategorisiert für einfaches Durchsuchen und Finden.

## Jobkategorien

- Frontend-Entwickler
- Backend-Entwickler
- Full-Stack-Entwickler
- Mobile Entwickler
- DevOps/SRE
- Data Engineer/Wissenschaftler
- ML/AI-Ingenieur
- Produktmanager
- Designer
- QA/Testing

## Verwendung

1. **Jobs durchsuchen**: Navigieren Sie zu einem bestimmten Datumsordner, um den Bericht anzuzeigen
2. **Jobs finden**: Klicken Sie auf einen Jobtitel, um direkt zur Originalanzeige zu gelangen
3. **Nach Kategorie filtern**: Jobs sind nach Kategorien gruppiert für schnelle Navigation

## Aktualisierungsfrequenz

- Automatisch jeden Montag aktualisiert
- Bei jeder Aktualisierung wird eine neue Berichtsdatei generiert
