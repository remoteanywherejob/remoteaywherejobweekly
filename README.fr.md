## À propos

[Remote Anywhere Job](https://www.remoteanywherejob.com) est une plateforme qui recueille des emplois logiciels à distance du monde entier.

Ce dépôt agrège automatiquement les dernières offres d'emploi logicielles à distance chaque semaine, organisées par catégorie pour une navigation et une recherche facilitées.

## Catégories d'Emplois

- Développeur Frontend
- Développeur Backend
- Développeur Full Stack
- Développeur Mobile
- DevOps/SRE
- Ingénieur Data/Scientifique de Données
- Ingénieur ML/AI
- Chef de Produit
- Designer
- QA/Test

## Comment Utiliser

1. **Parcourir les Emplois**: Accédez au dossier d'une date spécifique pour consulter le rapport
2. **Trouver des Emplois**: Cliquez sur le titre d'un emploi pour accéder directement à l'offre originale
3. **Filtrer par Catégorie**: Les emplois sont regroupés par catégorie pour une navigation rapide

## Fréquence de Mise à Jour

- Mis à jour automatiquement chaque lundi
- Un nouveau fichier de rapport est généré à chaque mise à jour
